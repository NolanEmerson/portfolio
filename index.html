My Works

Ding

Full meal planning web app

Ding is a full web application designed to simplify the meal planning process for the indecisive individual. It features a straightforward account registration, after which you gain immediate access to ingredients, nutritional information, and step-by-step instructions for thousands of recipes.

Ding was completed with a team of 4 developers. My main contributions were on the front end, developing the major functionality and routing of all pages and components, as well as the application's visual design and responsiveness.

The main technologies I used were React.js, React Router, Axios, CSS3, Materialize CSS, and PHP.

trailMix

Hiking and trail finder

trailMix is a hiking and trail finding application originally created during a 48-hour hackathon. It utilizes information from multiple APIs - Google Maps, Google Places, Google Geolocation, Google Directions, Hiking Project, and Dark Sky - to assist the user in finding trails and planning their trip.

My main contributions to the project were adding Google Places Autocomplete to the search bar, properly displaying trail information from the Hiking Project API, and aiding with devlopment of the weather component.

The main technologies I used were React.js, React Router, React Redux, Axios, Hiking Project API, Google Places API, Dark Sky API, and CSS3.
